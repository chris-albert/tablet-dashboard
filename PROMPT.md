# Description
A dashboard app for a Fire 7 tabled to display heads up information

# Technologies
 - React Typescript
 - Tanstack router
 - Tailwind
 - pnpm
 - Turborepo

# Display Info

 - SF Muni estimated times
  - M line inbound to Embarcadero from the Stonestown stop
  - 29 Bus to the Presideo from the Stonestown stop
 - Weather
  - 5 day weather forcast from SF areacode: 94132
